1. You can use the generated a.exe to run directly and enter the score according to the prompt (Need to decompress); An executable file can also be generated by "g++ main.cpp fraction.cpp -o a", and an exe file can be generated, refer to the picture "make_exe".
2. The test folder gives four test examples, including numerator 0, denominator 0, unreduced differentiation simplified case and general case. Other samples can also be tested.
3. My default constructor is to change the numerator and denominator to 1, you can also try other default constructor values by changing the code.
